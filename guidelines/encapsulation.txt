Use this algorithm to determine if a function (func) should be a method of a class (Cls).

if (func needs to be virtual)
{
    MEMBER.
}
else if (func is operator<< or operator>>)
{
    NON-member.
    if (func requires non-public access)
    {
        FRIEND.
    }
}
else if (func requires type conversion on left-most arg)
{
    NON-member.
    if (func requires non-public access)
    {
        FRIEND.
    }
}
else if (func requires non-public access)
{
    MEMBER.
}
else
{
    NON-member.
}
